# Database Migrations

This directory contains SQL migrations for the Golf Score Tracker database.

## How to Run Migrations

1. Go to your Supabase project dashboard
2. Navigate to **SQL Editor**
3. Click "New query"
4. Copy the SQL from the migration file
5. Paste it into the query editor
6. Click "Run"

## Migration Files

Each migration file is named with the format: `YYYYMMDD_description.sql`

Run migrations in order by date, oldest first.
